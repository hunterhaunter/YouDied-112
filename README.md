# You Died

[![CurseForge](https://cf.way2muchnoise.eu/582464.svg)](https://www.curseforge.com/minecraft/mc-mods/you-died)

Adds a Dark Souls inspired death screen to Minecraft.

When you die, the screen fades to a **"YOU DIED"** title with a sound sting, then fades into the normal respawn screen.

A 1.12.2 backport of [You Died](https://www.curseforge.com/minecraft/mc-mods/you-died) by Gory_Moon.

Requires Minecraft 1.12.2 and Forge 14.23.5.2860+. Client-side only.
