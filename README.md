# You Died

Adds a Dark Souls inspired death screen to Minecraft.

When you die, the screen fades to a **"YOU DIED"** title with a sound sting, then fades into the normal respawn screen.

A 1.12.2 Forge backport of [You Died](https://www.curseforge.com/minecraft/mc-mods/you-died) by Gory_Moon. There was no 1.12.2 build, so this brings it to legacy Forge.

## Install

- Minecraft 1.12.2
- Forge 14.23.5.2860 or newer
- Put the jar in your `mods` folder

Client-side only. Works on any server with nothing installed server-side.
